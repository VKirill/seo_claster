## Что такое Temperature?

Temperature контролирует "креативность" и случайность ответов модели:
- Низкая (0.0) = детерминированные, предсказуемые ответы
- Высокая (1.5) = креативные, разнообразные ответы

## Официальные рекомендации DeepSeek:

┌─────────────────────────────┬──────────────┬─────────────────┐
│ Use Case                    │ Temperature  │ Описание        │
├─────────────────────────────┼──────────────┼─────────────────┤
│ Coding / Math               │ 0.0          │ Точные расчёты  │
│ Data Cleaning / Analysis    │ 1.0          │ Анализ данных   │
│ General Conversation        │ 1.3          │ Общение         │
│ Translation                 │ 1.3          │ Перевод         │
│ Creative Writing / Poetry   │ 1.5          │ Творчество      │
└─────────────────────────────┴──────────────┴─────────────────┘

## Наш выбор: Temperature = 1.0

Используется в: seo_analyzer/analysis/deepseek_api_client.py

Причина:
✅ Задача = "Data Analysis" (анализ поисковых запросов)
✅ Нужен баланс между точностью и контекстом
✅ Не требуется абсолютный детерминизм (как в коде)
✅ Учитываются особенности ниши

## Когда менять Temperature?

### Temperature = 0.0 (детерминизм)
Если нужны абсолютно одинаковые результаты при повторных запросах:
- Расчёт конкретной формулы
- Математические вычисления
- Проверка корректности работы

### Temperature = 1.3 (больше разнообразия)
Если нужна большая вариативность:
- Генерация идей для контента
- Перефразирование текстов
- Анализ sentiment запросов

### Temperature = 1.5 (максимальная креативность)
Если нужна креативность:
- Генерация названий для продуктов
- Создание рекламных слоганов
- Креативный копирайтинг

## Как изменить Temperature?

1. Открыть: seo_analyzer/analysis/deepseek_api_client.py
2. Найти строку: "temperature": 1.0
3. Изменить значение на нужное (от 0.0 до 2.0)

Пример:
```python
payload = {
    "model": self.MODEL,
    "messages": [...],
    "temperature": 0.0,  # <-- Изменить здесь
    "max_tokens": 200
}
```

## Эксперименты с Temperature

Для тестирования разных значений:

```python
from seo_analyzer.analysis import DeepSeekAPIClient

client = DeepSeekAPIClient(api_key="your_key")

# Изменить temperature в коде или передать как параметр

```

## Рекомендации

✅ DO:
- Использовать 1.0 для анализа данных (текущий кейс)
- Использовать 0.0 для математических расчётов
- Тестировать разные значения для своей задачи

❌ DON'T:
- Не использовать 0.0 для анализа текста (слишком жёстко)
- Не использовать 1.5 для точных вычислений (слишком креативно)
- Не выставлять > 2.0 (нестабильные результаты)
